# Buy Later Goblin

Shopping list app with AI-powered item breakdown. Add items manually or paste a recipe/meal plan to auto-generate a categorized shopping list.

**Live:** https://dan-valang.github.io/unnamed-project/

## Dev

```bash
cd buy-later-goblin
npm install
npm run dev      # Vite dev server
npm run build    # tsc + vite build
```
